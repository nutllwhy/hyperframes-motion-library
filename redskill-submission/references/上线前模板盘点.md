# 视频动效系统上线前模板盘点

更新日期：2026-06-30

## 当前模板总览

系统目前登记 `14` 个可用模板，覆盖两条主线：

1. 数据与对比：适合用数据制造可信度、节奏点和视觉冲击。
2. 知识讲解：适合口播、教程、观点类视频里叠加解释、步骤和重点提示。

## 已有数据类模板

| 模板 | 用途 | 推荐场景 |
| --- | --- | --- |
| 柱状图增长 | 多项数据增长或对比 | 业绩、榜单、多维指标 |
| 数字计数器 | 单个关键数字放大呈现 | 里程碑、KPI、累计数据 |
| 折线绘制 | 趋势随时间展开 | 增长曲线、周期变化 |
| 数据脉冲 | 大数字 + 趋势说明 | 爆款增长、阶段性成果 |
| 双数据对比 | 两组数据横向对照 | 城市、产品、人群、方案对比 |
| 状态转折 | 两种状态章节式切换 | 过去 vs 现在、常规 vs 异常 |
| 极值刻度 | 一个极值数据强冲击呈现 | 风险指数、湿度、评分、占比 |

## 新增知识类模板

| 模板 | 用途 | 推荐场景 |
| --- | --- | --- |
| 概念解释卡 | 快速解释一个术语或方法 | “什么是 X”“这个概念怎么理解” |
| 三步流程 | 展示 SOP、教程、方法步骤 | “三步完成”“先做 A 再做 B” |
| 误区纠正 | 把错误认知切换成正确判断 | “很多人以为…其实…” |
| 重点标注 | 强调一句关键结论或金句 | 结论句、操作提醒、观点锚点 |
| 清单核对 | 逐条展示检查项或行动项 | “记住这三点”“上线前检查” |
| 时间线推进 | 展示阶段、版本、演变过程 | “从过去到现在”“三阶段变化” |
| 因果链路 | 解释原因、机制、结果 | “为什么会这样”“它怎么产生影响” |

## 发布时可以主打的能力

- 改文案、改数据，就能复用同一套动效。
- 同一个模板可输出透明 WebM，用来叠加到实拍、口播或剪辑画面上。
- 数据类模板负责“可信度和冲击力”，知识类模板负责“理解效率和记忆点”。
- 模板不是 PPT 页面，而是可渲染、可寻帧、可持续扩展的动效资产。

## 下一批值得补的模板

- 章节标题：适合给长视频分段，但要避免 PPT 页眉感。
- 证据卡片：引用一条数据、截图、观点来源，增强可信度。
- 关键词爆破：一个词从画面中心爆出，适合开头钩子和段落转折。
- 局部放大镜：圈出画面某处并放大说明，适合教程和产品演示。
