# 项目结构速览

## 核心文件

- README.md：项目介绍、GitHub 下载地址、在线演示页。
- SYSTEM.md：动效模板入库规范。
- AGENT_GUIDE.md：给 Agent 继续扩展系统的操作指南。
- catalog.json：模板库目录。
- app/：本地模板库界面。
- scripts/：本地检查、预览、渲染服务。
- templates/：每个动效模板的源码、设计说明和默认数据。

## RedSkill 投稿版保留的文件类型

- md：说明、规范、设计文档。
- html：动效模板源码和展示页。
- css：展示页样式。
- js：本地服务、渲染、检查脚本。
- json：模板目录、变量预设、package 配置。

## 没有放进投稿包的内容

- 视频样片：mp4、webm。
- 截图素材：png、jpg。
- 本地缓存、Git 信息和系统隐藏文件。

这些内容不是项目理解的必要条件。视频样片请通过在线演示页查看，完整项目请从 GitHub 下载。

在线演示页：https://nutllwhy.github.io/hyperframes-motion-library/

GitHub 仓库：https://github.com/nutllwhy/hyperframes-motion-library
