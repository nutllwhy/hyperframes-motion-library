# Mac 终端打字机视觉规范

## 用途

适合开发工具、AI 工作流、命令行教程和自动化流程演示。核心节奏是“窗口入场 → 命令逐字输入 → 执行反馈 → 第二条命令 → 完成提示”。

## 视觉方向

- 使用 macOS 深色终端窗口、三色窗口按钮和等宽字体。
- 命令、路径、成功反馈使用不同颜色分层，保持一眼可读。
- 光标闪烁和文字输入都必须由时间线控制，可任意寻帧。
- 背景只提供轻量网格与冷色光晕，不与终端内容竞争。

## 可复用变量

- 标题、副标题、两条命令、两条执行反馈。
- 背景色、终端强调色。

## 禁用项

- 不使用 CSS animation、随机字符和真实系统时间。
- 不把窗口坐标、缓动参数暴露为日常变量。
